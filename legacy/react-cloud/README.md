# Legacy: React-Cloud-Prototyp

`kdp-workbook-studio.legacy.jsx` ist die **unveränderte** Sicherung des
bisherigen React-/Cloud-Prototyps (im Auftrag als `Code.html` bereitgestellt;
der Inhalt ist tatsächlich eine JSX-Datei, keine HTML-Datei).

Diese Datei ist:

- **Produktwissen und Migrationsquelle** (Prompts, Workbook-Markup,
  KDP-Maße, UI-Flows, Qualitätsheuristiken),
- **nicht** Teil des Builds,
- **nicht** die Zielarchitektur,
- von den AGENTS.md-Regeln (400 LOC, 14 Zeilen/Funktion) als unveränderte
  Legacy-/Vendor-Datei ausgenommen.

Änderungen an dieser Datei sind verboten. Neue Erkenntnisse aus dem Altcode
werden in `docs/architecture/as-is-audit.md` und
`docs/migration/legacy-react-to-angular.md` dokumentiert.
